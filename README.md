# RoboSchema
Easily convert SQL Schemas to Salesforce

clone repo
npm install
roboSchema1.js: converts the SQL metadata to a CSV with Salesforce metadata
roboSchema2.js: converts the transformed CSV to Salesforce XML metadata you can deploy
